# Install

```
$ git clone path/to/repository
$ npm install
$ npm install -g babel // or sudo npm install -g babel
$ babel-node --harmony app.js
```

# Use

```bash
$ curl -H "Content-Type: application/json" -X POST -d '{"todo":"Acheter du lait"}' http://localhost:4000/api/todos
```
